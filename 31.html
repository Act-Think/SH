JavaScript - JavaScript에 대하여

  • HTML이 정보라는 멈춰 있는 이미지를 그리는 것이라면
    JavaScript는 정보라고 하는 그 이미지를
    마치 영화처럼 움직이게 한다. ex) 야간모드 버튼

    ∴ JavaScript는 웹페이지가 사용자와 상호작용할 수 있게 해준다.
    = JavaScript는 사용자와 상호작용을 하는 언어이다.

  • 웹브라우저는 한번 화면에 출력이 되면
    그 다음에는 자기 자신을 바꿀 수 있는 능력이 없다.
    그러나 그것을 가능하게 해주는 것이 JavaScript이다.

    ∴ JavaScript는 HTML을 제어하는 언어다.
    → 웹페이지를 훨씬 더 다이나믹하게 만들어준다.

  • 컴퓨터 프로그래밍 언어
    : 시간의 순서에 따라서 실행되어야 할 기능을
    프로그래밍 언어의 문법에 맞게 글로 적어 두는 방식

    HTML은 웹페이지를 묘사하는 목적의 언어이기 때문에
    시간의 순서에 따라서 무엇을 할 필요가 없고,
    따라서 시간의 순서에 따라 실행되는 기능을 갖고 있지 않다.

    그러나 JavaScript는 사용자와 상호작용하기 위해서 고안된 컴퓨터 언어이고
    그러기 위해서는 시간의 순서에 따라서 웹브라우저의 여러 기능이 실행되어야 하기 때문에
    프로그래밍이라고 하는 형태를 띠고 있다.

    ∴ JavaScript는 컴퓨터 언어이면서 동시에 컴퓨터 프로그래밍 언어라고 하고,
    HTML은 컴퓨터 언어이지만 컴퓨터 프로그래밍 언어라고 하지는 않는다.

  • JavaScript는 HTML 위에서 동작하는 언어다.
    ∴ JavaScript를 script 태그 안에 넣음으로써
    웹브라우저에게 그것이 JavaScript임을 알려준다.

  • 콘솔: 크롬 화면에서 마우스 오른쪽 버튼을 클릭한 후
    검사 버튼을 누르면 개발자 도구가 나온다.
    여기에 콘솔 탭이 있는데,
    이것을 이용하면 웹페이지 파일을 만들지 않고도
    JavaScript 코드를 즉석에서 실행할 수 있다.
